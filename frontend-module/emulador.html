Emulador de Dados da Cadeia Produtiva

Gera dados sintéticos (porém coerentes) das etapas do processo e minera cada evento como um bloco, usando a mesma lógica do cadastro manual. Útil para popular a blockchain rapidamente para demonstração e testes.

Cada lote percorre as etapas oficiais (até 7 blocos por lote). 0 = todos completam o fluxo. Ex.: 0.25 reprova ~1 em cada 4 na triagem.
Mesma semente gera exatamente os mesmos dados.
Mistura os lotes na cadeia (bom para testar o isolamento por lote na rastreabilidade).
